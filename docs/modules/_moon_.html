phase

  • phase(date?: DateTime): number
    • Defined in moon.ts:51

    Calculates the phase of the moon on the specified date.

         ============  ==============
     0 .. 6.99     New moon
     7 .. 13.99    First quarter
     14 .. 20.99   Full moon
     21 .. 27.99   Last quarter
     ============  ==============

    Parameters

    • Optional date: DateTime

      The date to calculate the phase for. Dates are always in the UTC timezone. If not specified then today's date is used.

    Returns number

    A number designating the phase.
