  • goldenHour(observer: Observer, date?: DateTime, direction?: SunDirection, tzinfo?: string): [DateTime, DateTime]
    • Defined in sun.ts:1056

    Returns the start and end times of the Golden Hour when the sun is traversing in the specified direction.

    This method uses the definition from PhotoPills i.e. the golden hour is when the sun is between 4 degrees below the horizon and 6 degrees above.

    Parameters

    • observer: Observer

      Observer to calculate the golden hour for

    • Optional date: DateTime

      Date for which to calculate the times. Default is today's date in the timezone tzinfo.

    • Default value direction: SunDirection = SunDirection.RISING

      Determines whether the time is for the sun rising or setting. Use SunDirection.RISING or SunDirection.SETTING.

    • Default value tzinfo: string = "utc"

      Timezone to return times in. Default is UTC.

    Returns [DateTime, DateTime]

    A tuple of the date and time at which the Golden Hour starts and ends.

timeAtElevation

  • timeAtElevation(observer: Observer, elevation: number, date?: DateTime, direction?: SunDirection, tzinfo?: string): DateTime
    • Defined in sun.ts:427

    Calculates the time when the sun is at the specified elevation on the specified date.

    Note: This method uses positive elevations for those above the horizon.

    Elevations greater than 90 degrees are converted to a setting sun
i.e. an elevation of 110 will calculate a setting sun at 70 degrees.

    Parameters

    • observer: Observer

      Observer to calculate for

    • elevation: number

      Elevation of the sun in degrees above the horizon to calculate for.

    • Optional date: DateTime

      Date to calculate for. Default is today's date in the timezone tzinfo.

    • Default value direction: SunDirection = SunDirection.RISING

      Determines whether the calculated time is for the sun rising or setting. Use SunDirection.RISING or SunDirection.SETTING. Default is rising.

    • Default value tzinfo: string = "utc"

      Timezone to return times in. Default is UTC.

    Returns DateTime

    Date and time at which the sun is at the specified elevation.
