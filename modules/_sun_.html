blueHour

  • blueHour(observer: Observer, __namedParameters?: { date: DateTime; direction: SunDirection; tzinfo: string }): [DateTime, DateTime]
    • Defined in src/astral/sun.ts:1076

    Returns the start and end times of the Blue Hour when the sun is traversing in the specified direction.

    This method uses the definition from PhotoPills i.e. the blue hour is when the sun is between 6 and 4 degrees below the horizon.

    throws

    ValueError if the sun does not transit the elevations -4 & -6 degrees

    Parameters

    • observer: Observer

      Observer to calculate the blue hour for

    • Default value __namedParameters: { date: DateTime; direction: SunDirection; tzinfo: string } = {}
      • date: DateTime

        Date for which to calculate the times. Default is today's date in the timezone tzinfo.

      • direction: SunDirection

        Determines whether the time is for the sun rising or setting. Use SunDirection.RISING or SunDirection.SETTING.

      • tzinfo: string

        Timezone to return times in. Default is UTC.

    Returns [DateTime, DateTime]

    A tuple of the date and time at which the Blue Hour starts and ends.

timeAtElevation

  • timeAtElevation(observer: Observer, elevation: number, date?: DateTime, direction?: SunDirection, tzinfo?: string): DateTime
    • Defined in src/astral/sun.ts:408

    Calculates the time when the sun is at the specified elevation on the specified date.

    Note: This method uses positive elevations for those above the horizon. Elevations greater than 90 degrees are converted to a setting sun i.e. an elevation of 110 will calculate the time for a setting sun at 70 degrees.

    Parameters

    • observer: Observer

      Observer to calculate for

    • elevation: number

      Elevation of the sun in degrees above the horizon to calculate for.

    • Optional date: DateTime

      Date to calculate for. Default is today's date in the timezone tzinfo.

    • Default value direction: SunDirection = SunDirection.RISING

      Determines whether the calculated time is for the sun rising or setting. Use SunDirection.RISING or SunDirection.SETTING. Default is rising.

    • Default value tzinfo: string = "utc"

      Timezone to return times in. Default is UTC.

    Returns DateTime

    Date and time at which the sun is at the specified elevation.
